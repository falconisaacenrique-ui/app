# Project Goal

**Build "LifeHub" — my personal daily-life command center.**

A local-first, installable web app (PWA) built with React + TypeScript + Vite. All data
stays on my device (localStorage) — no accounts, no server, works offline, installable
to my phone's home screen.

## Features (all in v1, equally weighted)

1. **Daily dashboard** — today at a glance: events, due tasks, upcoming reminders,
   habit progress, and monthly spending, with each card linking to its section.
2. **Calendar** — month view with events that can have times and colors.
3. **Notepad** — notes with pinning and auto-save.
4. **Reminders** — date/time reminders that fire browser notifications when due.
5. **Tasks** — to-do lists with due dates, priorities, and overdue highlighting.
6. **Habit tracker** — daily check-offs with streaks and a 7-day week view.
7. **Budget** — expense logging by category with a monthly budget and progress bar.

## Design — ultra-minimalist

- Mostly white/black with generous whitespace, thin typography, no decoration.
- No emojis anywhere — use a clean SVG line icon set (Lucide-style).
- Blue/indigo as the single accent color, used sparingly.
- Calm and distraction-free; dark mode follows system preference.
- Responsive: sidebar on desktop, bottom tab bar on mobile.
